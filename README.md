
# JavaScript Manzano

Projeto desenvolvido para praticar lógica de programação e algoritmos utilizando JavaScript com base nos exercícios do livro Manzano.

## 🚀 Objetivo

O objetivo deste projeto é aprimorar o raciocínio lógico, fortalecer a base em programação e desenvolver habilidades na construção de algoritmos através da resolução prática de exercícios.

## 🛠 Tecnologias utilizadas

- JavaScript
- Node.js
- Visual Studio Code

## ✨ Funcionalidades

- Resolução de exercícios de lógica
- Prática de algoritmos
- Estruturas condicionais
- Estruturas de repetição
- Manipulação de variáveis e operações matemáticas

## 📚 Conceitos praticados

- Lógica de programação
- Algoritmos
- Variáveis
- Operadores
- Condicionais
- Laços de repetição
- Entrada e saída de dados
- Raciocínio lógico
## 📌 Observação

Os exercícios foram desenvolvidos com foco educacional e prática contínua de programação utilizando JavaScript.
